# 🎮 Realm of the Bell

**Realm of the Bell** is a hybrid action game that blends traditional gameplay with gesture-based controls. Players can choose between standard keyboard input or controlling the game using real-time hand gestures through a webcam.

This dual control system creates a flexible and interactive experience. Whether you prefer familiar controls or want to explore a more experimental style of gameplay, the game allows you to switch between both modes seamlessly.

---

## ✨ Features

* 🎮 Dual control system (Keyboard + Gesture-based input)
* 🖐 Real-time hand tracking using a webcam
* ⚡ Smooth and responsive gameplay
* 🔄 Easy switching between control modes

---

## 🕹 How to Play

### 🔹 Normal Mode

Use keyboard controls to move and interact within the game.

### 🔹 Gesture Mode

* Enable your webcam
* Use hand gestures to perform actions such as shooting and interacting

---

## ⚠️ Important Notes

* Gesture mode requires a working webcam
* Please allow camera permissions when prompted
* Close other applications that may be using the camera

---

## 🔒 Privacy

The game uses your webcam only for real-time gesture detection.
**No images, videos, or personal data are recorded, stored, or transmitted.**
All processing happens locally on your device.

---

## 💡 Tips

* Use gesture mode in a well-lit environment
* Keep your hand clearly visible within the camera frame
